Stand Alone Complex

Nonsense systems have been reported to run on Debian, Ubuntu and CentOS.

You can use LuaRocks to install the nonsense pkg on Linux.

luarocks install pkg

If the installation fails make sure that you have singularity installed.

For help using pkg, including a list of commands, run:

$ pkg --help
